# Time-Off Microservice (Take-Home Submission)

This repository contains a defensive Time-Off Microservice built with NestJS and SQLite.

## Repository Structure

- `TRD.md` - Technical Requirement Document
- `SUBMISSION.md` - Submission checklist and verification notes
- `time-off-service/` - NestJS service implementation, tests, and coverage artifacts

## Tech Stack

- NestJS
- TypeORM
- SQLite
- Jest + Supertest

## Quick Start

From `time-off-service`:

```bash
npm install
npm run start:hcm
npm run start:dev
```

## Validation Commands

From `time-off-service`:

```bash
npm run build
npm run test:e2e
npm run test:e2e:cov
```

- Source code and documentation in this repository
